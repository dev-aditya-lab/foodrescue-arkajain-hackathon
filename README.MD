# Project -  Food Rescue - Tackling Food Waste with Technology
## Overview
Food Rescue is a web application designed to connect food donors (such as restaurants, grocery stores, and individuals) with local charities and food banks that can distribute surplus food to those in need. The platform aims to reduce food waste while addressing food insecurity in communities.
